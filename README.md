# 🌀 kotarsis

A multilingual collection of interactive web experiences built with Next.js.

🌐 Live: https://kotarsis.com

---

## About

kotarsis is a hub of independent digital worlds.

Each project explores a different idea, atmosphere, or emotional experience while sharing a common technical foundation and design philosophy.

Rather than focusing on utility, the projects focus on interaction, perception, storytelling, and immersion.

---

## Worlds

### 🔮 Oraculux
Digital mysticism and symbolic interaction.

### ⚡ Concentrator™
A satirical product landing page built around a fictional concentration-enhancing device.

### 🌿 Minimalism
An interactive exploration of simplicity, silence, and intentional living.

### ❤️ Redheart
An emotional manifesto presented as an interactive experience.

### 🕳️ Redacted
A psychological journey through loss, denial, and acceptance.

### 🚫 Censored
A satirical anti-corporate experience.

### 🎂 Birthday
Interactive birthday message generator.

---

## Features

- 12 languages (currently*)
- Server-side rendering (Next.js)
- Dynamic SEO metadata
- Open Graph support
- XML sitemap generation
- Shared i18n system
- Reusable UI architecture
- Responsive design

---

## Tech Stack

- Next.js
- React
- TypeScript
- SCSS
- App Router
- SSR
- Custom i18n

---

## Philosophy

People rarely remember interfaces.

They remember how something made them feel.

Kotarsis is an attempt to build experiences that leave a lasting impression rather than simply providing functionality.
